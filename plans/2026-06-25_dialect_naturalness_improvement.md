# 方言の自然さ改善プラン（文末表現）

作成日: 2026-06-25

## Context

チャットモードの応答は語彙（じょんのび・あんべ・たいそ等）はよく使えているが、
**文末（終助詞）が標準語の「〜よ」「〜ね」に倒れ、ペルソナ（妙高の70代女性）として
不自然**な箇所が残る。本プランは辞書とプロンプトの改修で文末の自然さを高める。

対象ファイル:
- `dialect_dict.txt` — 方言辞書
- `app.py` — `build_chat_prompt()` のチャット用 system prompt

---

## 問題の分析

実会話の「今」と「理想」を並べると、ズレはほぼ**文末の一点**に集約される。

| 今 | 理想 | 変換パターン |
|---|---|---|
| 気持ちいい**よ** | 気持ちいい**わね** / 気持ちいい**んだわ** | 〜よ → 〜わね / 〜んだわ |
| いい**ね** | いい**ねや** / いい**わね** | 〜ね → 〜ねや / 〜わね |
| じょんのびできる**よ** | じょんのびできる**わね** | 〜よ → 〜わね |
| 行きやすい**よ** | 行きやすい**わね** | 〜よ → 〜わね |
| 向いてる**よ** | 向いてる**わね** | 〜よ → 〜わね |
| 派手さは**ない**けど | 派手さは**ねえ**けど | ない → ねえ |

→ 課題は2点：**(1)「〜よ」の乱用**、**(2) 形容詞否定「ない→ねえ」の未適用**。

## 根本原因

1. 辞書の文法ルールに**平叙文の文末規定がない**（`〜ねぇ→〜ねや` はあるが、断定・教示の
   「〜よ」をどう言い換えるかが無い）。モデルは安全な標準語「よ／ね」に逃げる。
2. 会話例では `わね/んだわ/ねや` を多用しているのに**ルール化されていない**ため、
   プロンプトの「辞書に載っていない語尾は使ってはいけません」という制約が逆に働き、
   標準語「よ」に倒れる。
3. 否定「ない→ねえ」が語彙の `無い→ねぇ` しかなく、活用形（派手さはない・よくない等）に
   適用されない。

---

## 改修1: 辞書に「文末表現」セクションを追加

**ファイル:** `dialect_dict.txt`
**位置:** `## 文法・語法ルール` の直後（`## 語彙` より前）に最優先セクションとして挿入。

```
## 文末表現（終助詞）― このペルソナの自然さは文末で決まる・最重要 ―

平叙文を標準語の「〜よ」「〜ね」で終えない。70代女性らしく次へ言い換える。

断定・教示「〜よ」「〜だよ」「〜なんだよ」
  → 〜わね / 〜んだわ / 〜んだねや
  例）気持ちいいよ → 気持ちいいわね（気持ちいいんだわ）
  例）できるよ → できるわね（できるんだわ）
  例）行きやすいよ → 行きやすいわね（行きやすいんだわ）

同意・感想「〜ね」「〜いいね」「〜だね」
  → 〜ねや / 〜わね
  例）いいね → いいねや（いいわね）
  例）そうだね → そうだねや

丁寧「〜ですよ」「〜なのです」
  → 〜なんですわ / 〜なんだわね

形容詞・助動詞の否定「〜ない」→「〜ねえ」
  例）派手さはない → 派手さはねえ
  例）わからない → わからねえ ／ もったいない → もったいねえ

※「〜よ」で終わる文が続かないようにする。「わね」「ねや」「んだわ」を基本にする。
```

## 改修2: チャット用プロンプトの改善

**ファイル:** `app.py` / `build_chat_prompt()`

### (a) 文末ルールを明示（`【話し方の基本】` の直後あたりに追加）

```
【文末の言い方】― 重要 ―
あなたは妙高の70代女性です。文の終わりは標準語の「〜よ」「〜ね」で
終えず、やわらかい女性らしい妙高の言い回しを基本にしてください。
- 断定・教える時：「〜わね」「〜んだわ」「〜んだねや」
- 同意・感想：「〜ねや」「〜わね」
- 否定は「〜ない」ではなく「〜ねえ」
標準語の「〜よ」で終わる文が続かないよう、特に注意してください。
```

### (b) 過剰な制約を緩める

終助詞のバリエーションを殺している現行の制約を、語彙ルールは保ちつつ文末は自由にする。

```
# Before
辞書に載っていない方言・語尾・イントネーションは使ってはいけません。

# After
方言の語彙は辞書に載っているものを使ってください（辞書にない造語はしない）。
ただし文末の言い回し（わね・ねや・んだわ等）と語尾は、辞書の会話例に
ならって自然に使ってかまいません。
```

## 検証

1. `experiment/phase25/phrases.txt` に本プランの「理想」例（上表の左列＝標準語的応答の
   元になる文）を加え、翻訳モード相当で改修前後の出力を比較。
2. 特に確認する観点：
   - 文末が「よ」で終わる文の出現率が下がっているか
   - 「わね / ねや / んだわ」が自然に分散しているか
   - 否定が「ねえ」になっているか
3. チャットモードで実際に温泉の話題を一通り会話し、`_memo/` に before/after を記録。

## 留意点

- 語彙の造語抑制（辞書外の方言を作らない）は維持する。緩めるのは**文末のみ**。
- 「わね」を入れすぎると単調になるため、`ねや / んだわ / んだねや` と分散させる方針を
  辞書・プロンプト双方に明記済み。
- 翻訳モードのプロンプトは辞書を共有するため、辞書改修の恩恵を自動的に受ける。
